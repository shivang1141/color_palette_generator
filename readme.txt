Color Palette Generator

Description:
This Color Palette Generator is a user-friendly web application that enables users to generate random color palettes, useful for designers, artists, and developers. The application offers a dark and light theme, a customizable number of colors for the palette, and maintains a history of the last five palettes generated.

Key Features:
1. Generate random color palettes with a user-specified number of colors (3, 5, or 7).
2. Toggle between dark and light themes for the application.
3. Maintain a history of the last five generated palettes, with the option to clear the history.
4. Responsive design that works well on various devices and screen sizes.

Design:
The Color Palette Generator is designed with simplicity and ease of use in mind. It has a clean and minimalistic interface, with clearly labeled buttons and controls. The application utilizes CSS transitions and animations to provide a smooth and engaging user experience. It also has a responsive design that works well on various devices and screen sizes.
